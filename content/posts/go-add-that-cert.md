## 原來...
如果要 `自簽`，步驟大概會有這幾些：  

1. 產生 CA 憑證(.crt)
2. Server 自產 私鑰 及 CSR
3. Server 配合使用 CA 憑證簽出 Server 的憑證(.crt)
4. Client 連線時，使用認可的 CA 驗證 Server 憑證有效性

如果是 `使用認證機構` 頒發的憑證，那就直接從第 2 步走，  
第 4 步的話，通常使用國際認可機構頒發的憑證，  
電腦都會有預先設置，就可以不用再做這步。
